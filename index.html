Flexbox and a grid. Flexbox was created for layout in one dimension, which may be either a row or a column, as opposed to CSS Grid Layout, which was developed for layout in two dimensions. Grid was intended for simultaneous rows and columns in a two-dimensional arrangement.

In order to create different layouts depending on the size of the viewport, media queries are an essential component of responsive web design. However, they can also be used to detect other aspects of the environment in which your site is being viewed, such as whether the user is using a touchscreen instead of a mouse.

The size and dimensions of components on a web page may be determined using the CSS box model. Each HTML element is given a rectangular box that has four components: the content, padding, borders, and margins. When determining the size of an element, the browser starts with the width and height of the content, then adds the padding and border to that, and finally adds the margins. The total size of the box is the sum of all these parts. For example, if an element has a width of 200px, a padding of 20px, a border of 10px, and a margin of 30px, the total width of the element's box would be 290px (200px + 20px + 10px + 30px). This is useful for positioning and layout of elements in web pages.

Semantic HTML is a grammar that clarifies the various sections and page layouts in HTML to make it easier to understand. It improves the adaptability and information on web pages, which helps browsers and search engines understand the material.
